# Loneliness Embedding Project

This project explores the semantic structure of loneliness 
in Japanese and English corpora using word embeddings.

## Research Questions

- How is loneliness positioned in semantic space?
- How does it differ from isolation?
- How does meaning shift across time and culture?

## Structure

- data/ : raw and processed corpora
- scripts/ : preprocessing and embedding training
- notebooks/ : experiments and visualization
- results/ : outputs and figures
